Skills

HTML (HyperText Markup Language) es el lenguaje de marcado estándar utilizado para crear y estructurar el contenido en la web. Define la estructura básica de una página web mediante el uso de etiquetas, que representan diferentes tipos de contenido como texto, imágenes, enlaces, formularios, tablas, y otros elementos multimedia. CSS (Cascading Style Sheets) es un lenguaje de diseño utilizado para describir la presentación visual de una página web. Se encarga de definir cómo los elementos HTML se muestran en la pantalla o en otros medios. JavaScript es un lenguaje de programación dinámico y versátil que se utiliza principalmente para desarrollar aplicaciones web interactivas. Junto con HTML y CSS, es uno de los pilares fundamentales del desarrollo web. JavaScript permite añadir interactividad a las páginas, como manejar eventos (clics, desplazamiento, etc.), actualizar contenido en tiempo real, validar formularios, y mucho más. TypeScript es un lenguaje de programación basado en JavaScript que añade tipado estático y características avanzadas para mejorar el desarrollo de aplicaciones a gran escala. Fue desarrollado por Microsoft y es un superset de JavaScript, lo que significa que todo código JavaScript es también un código válido de TypeScript, pero con la opción de añadir tipos explícitos y otras herramientas útiles para el desarrollo.
